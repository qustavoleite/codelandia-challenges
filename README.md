### 👥 Codelândia 
É uma comunidade no discord, fundada pelo [Iuricode](https://github.com/iuricode), onde toda semana é liberado um desafio frontend em nível júnior no figma a fim da comunidade solucionar o desafio e aprimorar as suas habilidades. Participe da [Comunidade](https://discord.gg/2J4tNYgF).


### 🚀 Os desafios propostos são com as seguintes tecnologias:
- HTML
- CSS
- JAVASCRIPT
